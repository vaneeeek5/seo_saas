ADR-001
Project First Architecture
Статус
Approved
Контекст
Необходимо определить главную сущность системы.
Рассматривались варианты:
	•	Website
	•	Article
	•	Keyword
	•	Project
Решение
Главной сущностью платформы является Project.
Все остальные сущности принадлежат проекту.
Organization
    │
    └── Project
            ├── Website
            ├── Semantic
            ├── Content
            ├── Publications
            ├── Analytics
            └── Settings

Причины
Project является естественной границей бизнес-домена.
Большинство действий пользователя относятся не к статье, а к развитию сайта.
Последствия
Все документы должны содержать ProjectId.
Все API работают внутри проекта.
Удаление Project приводит к каскадному удалению зависимых данных.
