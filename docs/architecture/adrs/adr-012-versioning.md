ADR-012
Versioning
Контекст
Контент, промпты и API будут изменяться со временем.
Решение
Поддерживать версионирование:
	•	Content Asset;
	•	Prompt;
	•	API;
	•	Event Schema;
	•	Publication.
Последствия
Каждая публикация хранит ссылку на конкретную версию контента, а изменения не перезаписывают предыдущие версии.
