ADR-011
Immutable Event Log
Контекст
Для диагностики, аудита и повторной обработки требуется сохранять историю событий.
Решение
Все доменные события записываются в неизменяемый журнал событий.
Последствия
	•	возможно восстановление последовательности действий;
	•	упрощается аудит;
	•	появляется возможность повторной обработки событий при изменении логики.
