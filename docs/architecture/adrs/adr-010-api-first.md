ADR-010
API First
Контекст
Необходимо обеспечить возможность автоматизации и интеграции платформы.
Решение
Любая функциональность платформы должна быть доступна через API.
Веб-интерфейс является одним из клиентов этого API.
Последствия
Новые клиенты (CLI, мобильное приложение, внешние сервисы) используют те же контракты, что и веб-интерфейс.
